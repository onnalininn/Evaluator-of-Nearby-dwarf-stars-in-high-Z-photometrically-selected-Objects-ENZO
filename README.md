# Evaluator-of-Nearby-dwarf-stars-in-high-Z-photometrically-selected-Objects-ENZO

Simulation framework for estimating ultracool dwarf (UCD) contamination in high-redshift galaxy surveys observed at specifically area.

v1.0.0
Initial public release
Features:
* Photometric catalog generation
* Flux-to-magnitude conversion
* Monte Carlo simulations
* Photo-z estimations via EAzY package
* Color-selection analysis
* Number density calculations
* Support for Elf Owl, and Bobcat models
